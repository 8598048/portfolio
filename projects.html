Complete
Pi-hole DNS Sinkhole
Set up a Raspberry Pi 5 as a network-wide DNS sinkhole using Pi-hole v6. Flashed Raspberry Pi OS Lite, configured SSH remote access, and installed Pi-hole from the command line. The ISP router blocked DNS changes through its interface, so I disabled its DHCP server and used Pi-hole's built-in DHCP instead, covering every device on the network automatically. Ended up with 92,277 domains on the blocklist and full query logging across all clients in real time.

Windows Forensic Examination
Conducted a forensic investigation of three E01 disk images (suspect laptop + hotel kiosk) for a simulated criminal case. Used FTK Imager to verify image integrity via MD5/SHA1 hashes, Autopsy to analyze email artifacts, web search history, downloaded files, and browser cache, and Registry Explorer to identify the OS version and user activity. Documented a full suspect timeline, identified Guild operative communications, anti-forensics searches, and confirmed downloaded images via Zone.Identifier ADS.
